Projet S1

Site de Controverse

Découverte du développement web : création d'un site vitrine complet sans framework.

Voir le site

Contexte et Objectifs

Réalisé en équipe de 4 lors du premier semestre (S1), ce projet avait pour but de présenter une controverse actuelle sous la forme d'un site web attractif et informatif. La contrainte technique majeure était l'interdiction de tout framework CSS (type Bootstrap) pour garantir une maîtrise fondamentale du langage.

Apprentissage Fondamental

Conception responsive "à la main" via Flexbox et Media Queries, respect strict des standards W3C et de l'accessibilité de base.

Détails Techniques

HTML5 & Sémantique

Structure de page rigoureuse utilisant les balises sémantiques (header, nav, article, footer) pour optimiser le référencement et l'accessibilité.

CSS3 & Responsive

  • Mise en page via Flexbox
  • Design réactif (Desktop / Tablette / Mobile)
  • Code CSS modulaire et propre
  • Charte graphique cohérente
