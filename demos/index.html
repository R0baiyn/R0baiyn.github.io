Voir le site

Contexte et Méthodologie

Réalisé en équipe de 4, ce projet consiste en le développement complet (fullstack) d'un site web de démocratie participative. Nous avons adopté une méthode Agile SCRUM rigoureuse, avec des sprints de 3 semaines se concluant systématiquement par un rendez-vous client pour faire le point.

Communication Client & Product Owner

La communication avec le client durant les sprints passait exclusivement par notre Product Owner, garantissant un flux d'information clair et centralisé entre l'équipe de développement et les besoins métier.

Architecture Technique

Architecture MVC & SOLID

Implémentation stricte du modèle MVC (Modèle-Vue-Contrôleur) pour séparer la logique, les données et l'affichage. Le code respecte les principes SOLID pour assurer maintenabilité et évolutivité.

Fonctionnalités Clés

  • Architecture MVC complète
  • Principes SOLID respectés
  • Gestion de projet SCRUM
  • Communication via Product Owner
